How to Use the Ballot Site

This process is designed to be fair, simple, and ensure every registered user gets one unique, private ballot. Follow the steps below to participate in the draw.

1

Log In or Register

Enter your name and email on the login page. The name should match what the balot admin expects to be entered. If you are a new user, a profile is created immediately. If you are a returning user, your previous status is retrieved.

2

Check Status

The system automatically checks if you have already picked a ballot. If you have, you are immediately shown your previous result (Step 5).

3

View Available Ballots

If you have not picked, you will see a list of anonymous, shuffled ballots. You only get one chance, so select carefully! The number of remaining ballots is displayed prominently.

4

The Atomic Pick

When you click a ballot, the system marks the ballot as picked and assign it to you. This guarantees no one else can pick the same ballot, even milliseconds later.

5

Ballot Reveal

You are instantly redirected to the reveal page where your specific ballot result is shown. Your user session and profile are marked as 'picked' permanently.
